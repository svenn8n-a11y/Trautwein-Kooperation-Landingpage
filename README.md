# Trautwein-Kooperation-Landingpage

Pöppel-Kooperations-Landingpage für den Fuhrpark-/Flottenausbau mit **Trautwein Fahrzeugbau GmbH**.
Story: Trautwein baut die ALUCA-Fahrzeugeinrichtung, Pöppel liefert die Werkzeug-Grundausstattung –
ein fahr- und arbeitsfertiges Servicefahrzeug aus einer Hand.

- Absender/CI: R. Pöppel GmbH & Co. KG (CI-Rot #CC071E)
- Statischer One-Pager (`index.html`), Assets in `assets/`
- Lokal starten: `python3 -m http.server <PORT>` (reservierter Port siehe Localhost-Register)
- SEO/GEO: Meta-Tags, JSON-LD (Organization ×2, Service, FAQPage)

## Hinweis Bildrechte
Die Fotos unter `assets/img/` sind Referenzfotos der Trautwein Fahrzeugbau GmbH,
genutzt im Rahmen der Kooperation. Nutzungsfreigabe vor Livegang bestätigen.
